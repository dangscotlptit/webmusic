### Cài đặt
Cài đặt biến môi trường:
```
npm install
```

### Environment Variables
Tạo file `.env` và cài đặt biến:
| Tên           | Mô tả | Ví dụ |
|-----------------|-------------|---------|
| VITE_SERVER_URL | Server URL  | - http://localhost:8080      |

### Khởi chạy
Tại terminal, chạy command:

```
npm run dev
```
